# Customer Segmentation & Behavioral Analysis
### Alfido Tech Internship Task

## Project Overview
This project analyzes customer transaction data to identify customer segments, purchasing behavior, and churn risks. The analysis was conducted using Python with data analysis and visualization techniques.

The objective is to help Alfido Tech better understand customer behavior and design strategies to improve engagement and retention.

---

## Dataset
The dataset contains customer transaction information including:

- Customer ID
- Purchase Date
- Product Category
- Product Price
- Quantity
- Total Purchase Amount
- Payment Method
- Age
- Gender
- Returns
- Churn

Dataset Size:
- **250,000+ transactions**
- **Customer demographic and behavioral attributes**

## Customer Segmentation

Customers were segmented using **RFM Analysis**:

- **Recency** – How recently a customer purchased
- **Frequency** – How often they purchase
- **Monetary** – Total spending

Segments identified:

- Champions
- Loyal Customers
- Potential Loyalists
- At Risk Customers
- Lost Customers

---

## Key Insights

- Repeat customers generate the largest share of revenue.
- Certain product categories dominate transaction volume.
- Customers with longer purchase gaps show higher churn risk.
- Payment method preferences vary across segments.
- High-value customers require retention strategies.

## Business Recommendations

1. Implement a loyalty rewards program for repeat customers.
2. Launch re-engagement campaigns for inactive customers.
3. Provide personalized product recommendations.
4. Improve product descriptions and reviews to reduce returns.
5. Offer exclusive benefits for high-value customers.
